# rahulmalkani.in

Personal portfolio website of Rahul Malkani.

Built using TypeScript and modern web tooling.
Live at: https://www.rahulmalkani.in

---

## Running locally

```bash
git clone https://github.com/rahulmalkani/web-portfolio-src
cd rahulmalkani.in
npm install
npm run dev
```

Once the development server starts, the site should be available at [http://localhost:3000](http://localhost:3000).

Feel free to use this repository as a base and build on top of it.
